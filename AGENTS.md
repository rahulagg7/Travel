Project: Personal Travel Agent Reccomder WebApp for myself
Purpose: Whenever i want to travel i will run agent to it will ask me bunch of requirements. Based on that it will create a customizable iternary with my personalization for the trip.

Create a WHAT.md file with following details:
    - What is the purpose-type of trip? (business, vacation)
    - What is the budget for the trip? (max and min)
    - What is the duration of the trip?
    - What is the type of accomodation? (hotel, hostel, airbnb, etc.)
    - What is the type of transportation? (bus, car, train, flight, etc.)
    - What are the activites?

Create a PLAN.md file with following details:
    - Plan the journey from start to destination

Create a SPEC.md file with following details:
    - Recomment the tech stack for the project. - Frontend-Backend-Database-AI Agent

Create an Web App with following features:
    Cost
        - Estimate Cost of the trip
            - Estimate Cost of the accomodation
            - Estimate Cost of the transportation
            - Estimate Cost of the food
            - Estimate Cost of the activities
            - Estimate Cost of the other expenses
